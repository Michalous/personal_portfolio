Playing with Math.random()

t's very satisfying to watch computer compute stuff. Here are four little projects built with Math.random(). They are Monte Carlo integration, probabilities of rolling 2 dice, the birthday problem and shuffling the deck of cards.

Covid cases graph

This project pulls the data from the UK Government website and displays it as a bar chart using Plotly JS. To smooth out the spikes in the data I've calculated 7 day averages and displayed them alongside the actual data.

Bubble sort visualization

I first saw this visualization in one of the online CS lessons and decided to give it a try and build one for myself. It is a simplified animation only showing changes in order of elements omitting iterations where no swap is made.

Equinox countdown

Nice little project to show a countdown until next equinox. The biggest challenge here for me was to figure out how to display digits in the grid.
